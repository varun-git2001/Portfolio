Projects


AUTOMATED GATE SYSTEM USING IMAGE PROCESSING


The project utilizes image processing techniques to capture and analyze vehicle license plates as they approach the gate. After extracting and decoding the characters, they are matched against a database of authorized plates. If a match is found, the gate opens automatically, enabling seamless access.

AUTOMATED FIRE FIGHTING ROVER

In my current project, I am developing an Automated Fire Fighting Rover that integrates cutting-edge image processing and machine learning technologies. The rover is designed to detect fires using image processing algorithms, enabling it to swiftly navigate towards the flames while avoiding obstacles in its path.

Upon reaching the fire, the rover autonomously activates its water spraying mechanism to extinguish the flames effectively.
